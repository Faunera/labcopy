# ModuleEngine
Game engine library made with C++ modules.


# Requirements
Regarding libraries, see [NOTICE](./NOTICE.txt) for their sources and licenses. These libraries can be found in 'ext' folder.


Compilation:
- G++ 15.2.0
- Make
- Valgrind

Libraries:
- GLFW
- GLM
- TinyXML2
